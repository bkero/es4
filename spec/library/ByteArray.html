= ByteArray Objects = {{{ intrinsic final class ByteArray { } }}} Why is this class ''intrinsic''? It should be enough for it to be ''__ES4__''. The ''ByteArray'' class is new in 4th Edition. The ''ByteArray'' class in 4th Edition is compatible with the ''ByteArray'' class in ActionScript in the sense that it does not define any methods that conflict with methods defined by the latter (as of ActionScript 3 and Adobe Flex 2).
